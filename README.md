# Q and A

## サイト概要
Q & AアプリのMVP(Minimum Viable Product)を制作する。

### サイトテーマ
質問投稿サイト

### テーマを選んだ理由
Ruby on Railsの基礎学習としてCRUDアプリケーションの制作を行う。

### 機能
- 質問投稿
- 質問編集
- 質問削除
- 質問回答
- 質問一覧
- 回答編集
- 回答削除
- 回答一覧
